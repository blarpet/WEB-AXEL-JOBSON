Element
Ett element är allt mellan en start tag och en slut tag som något mellan en p och /p.
HTML
HTML är det standarda påläggs språket för webb design. Html är uppbyggt av olika element som förklarar för browsern hur saker skall se ut.
Id
Id så att säga är en identification för olika element. Id används för att välja en specifik style declaration i sitt lilla style sheet.
Klass
Klass eller class attributet är ett namn som sammankopplar en med likadant namn i css en fil.
Meta
Meta eller metadata visar data inom ett HTML dokument. Data menas att det är information och data taggen skall altid gå inom head elementet. Metadata används oftasts av olika sök motorer.
Responsive
Responsive menas att något anpassar sig beronede på vilken enhet man vissar hemsidan med. Till exempel om man är på mobil så ändras formatet att passa en mobils skärm.
HTTP
HTTP är sätt om olika klients skickar ett krav till webben som den sedan efter det får ett svar från webben.
HTTPS
HTTPS är en annan version av HTTP men en mer säker version av HTTP.
Data compression
Data compression är när man gömmer information med att använda mindre bits än den förra version av datan. Det finns 2 olika sätt att göra det och det är inom lossy och lossless compression.
CDN
CDN eller Content Delivery Network är en grupp av servrar som arbetar tillsammans för att skapa snabb nerladdnings och flyttning av information på webben.
Local Bootstrap
Local bootstrap är offline bootstrap som en boostrap.css fil inom Visual Studio Code
jQuery
jQuery är ett library som bootstrap som gör det att man behöver skriva mindre när man gör hemsidor med JavaScript
FlexBox
FlexBox är en Layout model som gör det lättare att skapa en flexible stuktur.
Grid
Grid Layouten ger ett grid-baserat layout system. Detta likt som FlexBox hälper att skapa hemsidor på ett lättare sätt.
Code snippets
Code snippets är en liten samlig av text eller kod som använda för att sätta in common kod och funktioner.
